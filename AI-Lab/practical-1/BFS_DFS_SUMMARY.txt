================================================================================
                        ğŸ“– ALGORITHM EXPLANATIONS
================================================================================

BFS (BREADTH-FIRST SEARCH):
  What: Explores graph level-by-level
  How: Uses Queue (FIFO) - First In, First Out
  Why: Guarantees shortest path in unweighted graphs
  When: Need optimal solution, close nodes first
  Example: Social network (friends, friends-of-friends...)

DFS (DEPTH-FIRST SEARCH):
  What: Explores as deep as possible before backtracking
  How: Uses Stack (LIFO) - Last In, First Out
  Why: Memory efficient, good for deep searches
  When: Need to explore all paths, tree traversal
  Example: Maze solving, puzzle solving

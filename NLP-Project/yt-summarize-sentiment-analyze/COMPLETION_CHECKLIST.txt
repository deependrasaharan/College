USAGE EXAMPLES
═══════════════════════════════════════════════════════════════════════════

Basic Commands:

1. Get help:
   python youtube_nlp_analyzer.py --help

2. Transcript only (no API key needed):
   python youtube_nlp_analyzer.py "YOUTUBE_URL" --transcript-only

3. Full summary (requires API key):
   python youtube_nlp_analyzer.py "YOUTUBE_URL"

4. With sentiment analysis:
   python youtube_nlp_analyzer.py "YOUTUBE_URL" --sentiment

5. Adjust temperature:
   python youtube_nlp_analyzer.py "YOUTUBE_URL" --temperature 1.2

6. Different language:
   python youtube_nlp_analyzer.py "YOUTUBE_URL" --language es

7. Custom summary length:
   python youtube_nlp_analyzer.py "YOUTUBE_URL" --max-tokens 1000

8. Use GPT-4:
   python youtube_nlp_analyzer.py "YOUTUBE_URL" --model gpt-4

Programmatic Usage:

from youtube_nlp_analyzer import YouTubeNLPAnalyzer

analyzer = YouTubeNLPAnalyzer()
results = analyzer.full_analysis(
    "https://www.youtube.com/watch?v=VIDEO_ID",
    temperature=0.7,
    include_sentiment=True
)

print(results['summary'])
print(results['sentiment'])
